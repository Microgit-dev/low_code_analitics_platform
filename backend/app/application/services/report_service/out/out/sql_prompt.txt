1. HTML-шаблон отчёта
2. Описание структуры базы данных: таблицы, поля, типы данных, связи
3. Текст запроса на отчёт

Задача:
- понять, какие данные нужны для заполнения отчёта
- учесть структуру таблиц в HTML-шаблоне

- использовать только те таблицы и колонки, которые описаны в структуре БД
- при необходимости использовать JOIN, GROUP BY, ORDER BY, агрегаты

Строгие правила ответа:
- верни только SQL-запрос
- без markdown
- без тройных кавычек
- без комментариев
- без пояснений
- только один SQL-запрос
- запрос должен быть SELECT-запросом или WITH ... SELECT
